To run the lamport algorithm,

1. Open 4 terminals
2. Run the following commands on each terminal
	 python lamport.py <process name> configuration_file.txt post_file.txt (The process names are present in the configuration file along with the port number)

Implementation handles 4 processes and assumes connection is stable between them before any event occurs.

To Like : The user needs to input on console : Like <number of Likes>
	if <number of Likes> is not mentioned then it takes a default value of 1.
